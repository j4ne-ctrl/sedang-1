# LANGKAH KERJA PROGRAM HITUNG FAKTORIAL
Program ini berfungsi untuk menghitung nilai faktorial dari angka yang dimasukkan oleh user.

Dalam program ini, user diminta untuk memasukkan angka, setelah itu didalam program membyat variable faktorial dengan nilai awal 1. Lalu program menggunakan perulangan for dari 1 sampai angka yang dimasukkan. Setiap perulangan, nilai faktorial dikalikan dengan angka ke-i, setelah perulangan selesai, program menampilkan hasil faktorial
